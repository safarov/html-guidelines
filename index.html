Инструменты

  • styletto — asset manager for css. Собирает css в один файл, компилирует stylus, жмет файл через csso или yui compressor и кодирует картинки в base64.
  • grunt — станок для сборки ресурсов.
  • grunt-styletto — плагин для grunt, добавляющий поддержку styletto.
  • jam-boilerplate — Базовый шаблок для верстки на основе html5-boilerplate с правками под SEO/аналитику и с готовыми build-файлами.
  • imgo — библиотека для обработки изображений под Mac OS X.
  • csscomb — сортировщик правил в css.

О стандартах

Данный список стандартов разрабатывается и поддерживается компанией JetStyle.

Любой сайт, производимый компанией JetStyle должен удовлетворять положениям данных стандартов, если в техническом задании на этот сайт не оговорено обратное.

Стандарты подлежат обязательному пересмотру не реже чем 1 раз в шесть месяцев.
